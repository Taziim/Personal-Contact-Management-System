###Box Model###
--there are two types of box model--
--box sizzing default to content box--
--content box the width and height you set for an element only include the content area. The padding and border are added to the width and height, which means the total size of the element will be larger than the width and height you specify--
--border box With this box model, the width and height you set for an element include the padding and border. This means the total size of the element will be exactly what you specify in the width and height, and it won't grow due to padding or border--
###inline element and block level element###

--An inline element does not start on a new line; it stays in the same line as the surrounding content--
